☁ HOW TO CONNECT GOOGLE DRIVE

To enable Save to Drive and Load from Drive, you need a free Google OAuth Client ID. Takes about 5 minutes.

  1. Go to console.cloud.google.com and sign in with your Google account
  2. Click Create Project → name it "OGBcards" → Create
  3. Left menu → APIs & Services → Library → search Google Drive API → Enable
  4. Left menu → OAuth consent screen → External → fill in app name "OGBcards" and your email → Save
  5. Left menu → Credentials → Create Credentials → OAuth Client ID
  6. Application type: Web application · Name: OGBcards
  7. Under Authorised JavaScript origins add: https://YOUR-USERNAME.github.io
  8. Click Create → copy the Client ID
  9. Open ogbcards.html in a text editor, find the line:
    const GDRIVE_CLIENT_ID = '';
    and paste your Client ID between the quotes
  10. Save and re-upload to GitHub Pages — Drive sync now works everywhere

Your data saves as ogbcards-data.json in your Google Drive root. Only you can access it.
